> 《番茄（秋季）专属数据集报告》构建了一个**面向8米单棚秋季樱桃番茄的高精度多源数据体系**，实现了环境—生理—农艺数据的完整闭环，为后续模型校准、精准管理与智慧农业系统开发提供了坚实的数据基础。

------

## 一、总体定位与核心价值

本报告由是“樱桃番茄数字化管理配套技术”项目的数据基础文件。
 其主要作用是：

> 通过系统采集和整理**环境因子 + 生理参数 + 农艺管理数据**，建立一个**高精度、秋季专属的数据集**，用于支持模型训练、参数校准和智能化决策。

数据集总量达：

- **环境监测数据**：11.5万条（10分钟–1小时分辨率）
- **生理农艺数据**：480组
- **管理措施数据**：86条
   数据完整率达 **98.5% 以上**，为非自动化8米单棚番茄种植提供了完整的数字化样本。

------

## 二、数据采集设计与范围

### 1. 采集目标

面向**秋季8米单棚樱桃番茄**种植，采集环境、作物生理及农艺操作全周期数据，用于：

- 构建智能管理模型；
- 支撑温湿光CO₂调控；
- 指导水肥灌溉；
- 验证产量与病害预测。

### 2. 采集范围

- **空间维度**：单棚划分为北、中、南3个监测区；每区分上层、冠层、根区3层采样，空间覆盖全面；
- **作物维度**：选用品种“龙珠”，无限生长型，适合秋季短日照环境；
- **时间维度**：覆盖2025年8月–10月全生育期（123天）。

### 3. 采集方法

- **环境数据**：多点传感器同步采样（温湿度、光照、CO₂、土壤湿度与EC值），高频率记录；
- **生理与农艺数据**：
  - 光合仪测定净光合速率（27–32 μmol/m²·s）；
  - 叶面积仪测量LAI；
  - 人工记录整枝、灌溉、施肥、病虫害；
  - 实测果实糖度（7.8–8.2 °Brix）。

## 三、核心技术参数

| 数据类型      | 时间分辨率 | 指标                          |
| ------------- | ---------- | ----------------------------- |
| 空气温湿度    | 10分钟     | 白天18–23℃、夜间14–17℃        |

| CO₂浓度       | 30分钟     | 400–750 ppm（增施后）         |
| 土壤湿度与EC  | 1小时      | 50–75%FC，1.0–1.7 mS/cm       |
| 光合/LAI/株高 | 日–周尺度  | 动态采样与周均值记录          |

这些参数构成了番茄秋季生长的**高频多维数据网格**，可直接输入模型进行训练。

## 四、数据质量控制体系

报告的数据质量管控极为严谨，采用了“三层防线”机制：

1. **传感器校准与维护**
   - 温度模型透光率 α=0.72–0.78；
   - 土壤传感器IP68防水型替换，误差<2%；
   - 校准周期缩短为10天。
2. **数据完整性保障**
   - 三重冗余采集（北-中-南三区）；
   - 异常值通过“3σ原则”剔除；
   - 缺失数据线性插值补全并标记。
3. **数据验证与匹配**
   - 每周人工比对实测（误差超5%即重新标定）；
   - 建立生理数据与环境数据的对应关系验证（如Pn与光照强度、CO₂浓度一致性）。

数据完整率高达 **98.5%**，保证了可溯源性与科研可靠性。

## 五、数据分析与发现

### 1. 环境数据分析

- **温湿度**：夜间温控在14–17℃显著降低呼吸消耗10%–13%；
- **光照与CO₂**：秋季短日照（8–10小时），通过补光维持12小时临界日照；CO₂增施至750ppm后光合速率提升23%，与模型预测吻合；
- **土壤数据**：EC随施肥呈周期性波动（1.0→1.7mS/cm），反映钾需求峰值。

### 2. 生理与农艺数据分析

- 品种“龙珠”秋季生理参数明确：坐果率66.7%，单果重42g；
- 秋季光饱和点较夏季降低200–300μmol/m²/s；
- 干物质分配向果实比例高达68%，说明能量利用更集中；
- 灌溉与施肥时序高度标准化，可复制性强。

### 3. 数据关联分析

建立了**环境-生理-产量三维关联模型**：

- 每升高昼夜温差1℃，糖度上升0.25–0.35°Brix；
- EC从1.8→2.3mS/cm，单果重增7–9%；
- 光照降低时，CO₂浓度与Pn呈显著正相关（R²>0.9）。

这些结果为机器学习模型提供了清晰的输入-输出关系。
