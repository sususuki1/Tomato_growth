《番茄（秋季）生长模型技术手册》是秋季8米单棚番茄数字化管理系统的**技术核心文档**，实现了生长预测、水肥调控与病虫害预警的统一模拟，具备**高精度、可扩展、可复制**三大特性，是实现“AI辅助温室管理”的关键桥梁。

------

## 一、文档定位与总体目标

该手册是为**秋季8米单棚樱桃番茄**开发的**生长模拟与管理决策系统技术文件**，主要作用是：

> 将“机理模型 + 校准参数 + 数据集”三者整合为**可运行的模型程序体系**，指导生长预测、水肥管理与病虫害预警，实现低成本数字化管理。

手册既是技术标准，也是运行指南，内容兼具**科学严谨性、可复现性与实操性**。

------

## 二、模型总体框架与核心功能

### 1. 三大核心功能

模型整合三大模块，形成**全生命周期的智能管理链**：

| 模块               | 功能                                        | 输出                                           |
| ------------------ | ------------------------------------------- | ---------------------------------------------- |
| **生长模拟模型**   | 模拟从定植到采收的积温、LAI、干物质动态变化 | 生育进程预测（开花期、采收期）、叶面积指数变化 |
| **水肥运输模型**   | 模拟土壤EC、钾吸收效率、根系吸水效率        | EC值监控、钾吸收量、水分预警与调控建议         |
| **病虫害预警模型** | 预测灰霉病与白粉虱高发时段                  | 风险指数、预警提示、建议措施                   |

三者通过**时序联动与参数共享**协同运行：

- 生长模型提供当前生育阶段；
- 水肥与病虫害模块依据阶段自动启用；
- 各模块共享温湿度等核心环境参数；
- 模拟结果形成闭环反馈（例如：干物质→钾需求）。

------

### 2. 参数来源与科学性保障

所有参数**均来自已验证的三类文档**：

- 《番茄（秋季）生长模型原始校准报告》：温度、CO₂、水分、病害参数；
- 《番茄（秋季）专属数据集报告》：温控与水肥阈值；
- 《番茄生长模型参数表》：钾吸收、光合速率等生理参数。

不使用经验估算或外部季节数据，确保模型**精准适配“秋季8米单棚”环境**。

## 三、核心技术体系与关键参数

### （一）温度与积温机制

- **基点温度（T_base）**：10℃
- **积温修正系数（GDD_corr）**：开花期0.92、果实期0.95、采收期0.98
- **最优坐果温度（opt_temp）**：白天23℃、夜间17℃
   → 若夜温低于15℃或高于28℃，坐果率下降≥30%。

### （二）CO₂与光合模型

- **光合量子效率（φ）**：0.037 molCO₂/molPAR
- **暗呼吸速率（Rd）**：0.023 g干物质/g·h
- **最优CO₂浓度**：750 ppm（高于800 ppm增益<5%）

→ 模拟光照与CO₂变化对干物质累积的影响，自动调整夜温对呼吸的修正。

### （三）水肥动态模型

- **EC阈值**：1.8–2.3 mS/cm
- **钾吸收峰值**：1.0 g/株·d（果实膨大期）
- **土壤最优含水量**：65–70% FC
- **吸收速率方程**：基于Michaelis-Menten动力学（Vmax=7.5 g/株·周，Km=2.0）

→ 当EC>2.3 mS/cm，根系吸水效率下降超20%，模型将提示“减少盐分输入”。

### （四）病虫害预警机制

- **灰霉病**：湿度≥80%、温度14–18℃ → 指数型风险增长；
- **白粉虱**：适宜温度20–25℃ → 世代周期12天；
- **预警准确率**：≥88%，误差<3天。

## 四、模型运行逻辑与数据要求

### 1. 输入数据

分为两类：

- **气象数据**：日均温、夜温、夜湿度、PAR光照量；
- **种植数据**：定植日期、棚尺寸、灌溉频率、品种信息。
   要求格式为CSV/Excel，字段命名规范（如`avg_temp`, `night_humidity`等），无缺失或异常值。

### 2. 输出数据

输出分为三类：

| 模块       | 输出内容                    | 示例                        |
| ---------- | --------------------------- | --------------------------- |
| 生长发育   | 开花日、采收期、LAI、干物质 | `2025-10-10 开花`           |
| 水肥管理   | 土壤EC、钾吸收量、水分效率  | `EC=2.15，建议减少盐分输入` |
| 病虫害预警 | 灰霉病指数、白粉虱高发期    | `风险指数=87，高风险`       |

同时支持生成三类趋势图（LAI、水肥、病害），便于可视化分析。

## 六、使用与扩展说明

### （1）操作环境

- Python ≥ 3.7（推荐3.7–3.9）；
- 必装库：`numpy 1.19.0`、`pandas 1.2.0`；
- 可选库：`matplotlib 3.3.4`（生成图表）。

### （2）使用流程

1. 导入依赖库；
2. 准备CSV输入数据；
3. 校对参数（若为标准8米棚无需改动）；
4. 执行运行命令；
5. 查看输出文件与图表；
6. 根据`management_suggestion`执行管理操作。

### （3）异常与误差处理

提供详细排查策略：

- 参数错设（T_base、GDD_corr误配）；
- 输入文件路径错误；
- 数据缺失或格式错误；
- 灰霉病预警低准确度的修正方法；
- EC、K吸收偏差修复流程。

### （4）模型扩展能力

- **棚室扩展**：支持10m、12m棚，通过面积比例自动调整灌溉与肥量；
- **实时数据接入**：可对接传感器与气象站，实现动态模拟；
- **功能扩展**：支持Scipy参数优化、结果持久化、品种参数库加载。
